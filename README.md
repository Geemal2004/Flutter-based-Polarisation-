# OptiCalc Mobile

OptiCalc Mobile is a real-time optical polarization simulator built with Flutter.
It models Stokes vectors passing through a configurable chain of Mueller-matrix
optical elements (linear polarizers, retarders, and optical rotators).

## Features

- Preset and custom Stokes vector input
- Real-time Mueller matrix chain evaluation
- Derived polarization metrics (DOP, azimuth, ellipticity, Poincaré coordinates)

## Getting Started

1. Install Flutter (>=3.22) and Dart (>=3.3).
2. Run `flutter pub get`.
3. Launch with `flutter run`.
